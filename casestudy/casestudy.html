TRIGGERS

Trigger is the hover on the project card

RULES

Once a pointer is hovered over the drexel card, in 250ms, the card will scale up by 5% and the details card would pop up with a golden background and cover the original card completely.


The details card would have 3 elements and each of them have a different rule. these are:

  1. Main Heading: Drexel Athletics:the Main heading opacity and scale will change from 0-100% with a transition delay of 100ms.

  2. Brief: opacity and size will change from 0-100% with a transition delay of 200ms.

  3. Software tags:The software tag opacity and size will change from 0-100% with a transition delay of 250ms.

  4. Beta Addition: I added a View project button which will change from opacity 0 to 100% in 650ms. And a modal that will open in 250ms and display the basic details of the project along with an option to view full project. The buttons also have hover states on and change background color in 500ms when hovered over.

FEEDBACK

The Feedback would be the visibilty of the details Card, and the visibility of the modal when the user clicks: View Project Preview. The buttons also have a hover effect applied that changes their background color when hovered.

LOOPS

There are no loops

MODES

The mode is the modal, it is active when someone clicks on View Project Preview Button, it is unactive when someone uses the cross button on the top right of the modal to close the modal.
