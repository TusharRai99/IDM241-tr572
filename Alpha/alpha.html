Microinteraction Alpha

TRIGGERS

Trigger would be a hover on the project card

RULES

Once a pointer is hovered the original yellow screen will scale up by 5 percent. Also, a details card would pop up with a blue background and cover the original card almost completely, with a transition where the opacity of the details card changes from 0 to 100 percent in 250ms. The details card has 3 elements and each of them have a different rule. these are:

  1. Main Heading: Drexel Athletics: On hover over the the Main Project card, the Main heading opacity and size will change from 0-100% in 300ms
  2. Brief: On hover over the the Main Project card, the Brief opacity and size will change from 0-100% in 350ms
  3. Software tags: On hover over the the Main Project card, the software tag opacity and size will change from 0-100% in 400ms

FEEDBACK

The Feedback would be the visibilty of the details Card

LOOPS

There would be no loops.
